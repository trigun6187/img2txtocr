# 🔤 Free Image to Text Converter (OCR Tool)

A completely free, client-side online OCR tool for extracting text from images and PDFs. No signup required, no API keys, 100% private - everything runs in your browser!

## ✨ Features

- ⚡ **Instant Processing** - Extract text from images instantly
- 🔒 **Private & Secure** - Files never leave your browser (client-side processing)
- 🌍 **100+ Languages** - Support for English, Chinese, Japanese, Korean, German, French, Spanish, Portuguese, Russian and more
- 💻 **Works Offline** - No internet connection required after loading
- 📱 **Mobile Friendly** - Works on phones and tablets

## 🚀 How to Use

1. Drag & drop your image or PDF file (or click to browse)
2. Select the language of the text in the image
3. Click "Extract Text" button
4. Copy the extracted text or download as .txt file

## 🔧 Technical Details

- **Technology**: Tesseract.js - Open source OCR engine running entirely in browser
- **Hosting**: GitHub Pages (FREE)
- **No Backend Required** - Everything runs client-side
- **No API Keys Needed** - Uses open-source libraries

## 📊 Supported Formats

- Images: JPG, PNG, GIF, BMP, WEBP
- Documents: PDF files
